=====[LOADING DATA]===========================================

LOAD DATA INFILE => THIS SQL COMMAND READS 
THE ROWS FROM A TEXT FILE INTO AN EXISTING 
MYSQL TABLE WITHIN A DB AT HIGH SPEED


===[LOADING A TABLE WITH DATA FROM A CSV FILE]===========

LOAD DATA INFILE '/path/to/csv/file.csv'
INTO TABLE my_table_name
FIELDS TERMINATED BY ','
ENCLOSED BY '"'
LINES TERMINATED BY '\n'
IGNORE 1 ROWS;


"IGNORE 1 ROWS" => THIS TELLS MYSQL TO IGNORE 
THE FIRST ROW OF DATA TO PREVENT THE HEADERS 
FROM THE CSV FILE FROM BEING IMPORTED INTO 
THE MYSQL TABLE
